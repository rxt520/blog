16.使用冒泡排序

public class SortBymaopao {
    public static void main(String[] args) {
        int[] arr = {1,3,5,2,4,7,8};
        sortBymaoPao(arr);
        System.out.println(Arrays.toString(arr));
    }
    public static void sortBymaoPao(int[] arr){
        for (int i = 0; i < arr.length; i++) {
            for (int j = 0; j < arr.length-i-1; j++) {
                if (arr[j]>arr[j+1]){
                    int temp;
                    temp = arr[j];
                    arr[j] = arr[j+1];
                    arr[j+1] = temp;
                }
            }
        }
    }
}

19.二分查找

package com.rxt.tsort;

/**
 * @author Ruan.X.T.
 * @create 2020-02-13-22:09
 */
public class tSort {
    public static void main(String[] args) {
        int[] arr = {1,2,3,4,5,6,7,8,9};
        int num = 9;
        System.out.println(tfind(arr, num));
    }
    public static int tfind(int[] arr,int num){
        int head = 0;
        int last = arr.length-1;
        int mid=0;
        int n;
        while (head<=last){
            mid = (head+last)/2;
            if (arr[mid]==num){
                break;
            }else if (arr[mid]>num){
                last = mid-1;
            }else if (arr[mid]<num){
                head = mid + 1;
            }
        }
        return mid;
    }
}

30.static final abstract 用来修饰什么的？

  • abstract 可以修饰类和方法，抽象方法一定在抽象类里，抽象类不一定有抽象方法
  • static 表示静态，可以修饰属性，方法，代码块
  • final 修饰符，可以修饰变量，方法，类
    • final修饰变量会变成常量，一旦赋值不能改变，可以在初始化或者构造方法里赋值，只能在这两种方法里二选一，不能不为常量复制，final经常和static 一起使用
    • final修饰方法，被final修饰的方法将不能被其子类覆盖，保持方法的稳定不能被覆盖。
    • final修饰类，被final修饰的类将不能被继承，final类中的方法也都是final的。

31.抽象类和接口有哪些共同点和区别？

  • 共同点：
    • 不能实例化
    • 都可以定义未实现的方法
    • 都可以被继承
  • 不同点：
    • 抽象类有构造器，接口不能声明构造器
    • 接口可以被多继承

33.ArrayList、LinkedList、Vector三者的异同？

  • 同：三个类都是实现了List接口、存储有序、可重复的数据

  • 不同：

    • ArrayList是List接口的主要实现类；线程不安全，效率高。底层使用object[] elementData 存储数据
    • LinkedList 对于频繁的插入与删除使用此类效率较高，底层使用双向链表存储
    • Vector：是List接口的古老实现类，线程安全效率低。底层使用Object[ ] elementData 存储

  • ArrayList 底层 为elementData数组，1.7的时候初始化为容量为10的数组，1.8的时候初始化为0，当进行添加的时候才进行扩容为10.当添加元素超过10的时候需要进行扩容，容量为1.5倍

34.HashSet、LinkedHashSet、TreeSet的异同？

  • 同：存储无序、不可重复的数据
  • 异：
    • HashSet是Set接口的主要实现类、线程是不安全的；可以存储null值
    • LinkedHashSet 作为HashSet的子类，遍历他的内部数据时，可以按照添加顺序进行遍历
    • TreeSet：可以按照添加对象的指定属性，进行排序
